Fokus

O objetivo deste projeto foi criar um cronômetro funcional com a possibilidade de alternar entre períodos d e estudo e descanso, utilizando um design responsivo e interativo. A integração de música de fundo e a alternância visual de contexto enriquecem a experiência do usuário.

Fokus é um cronômetro interativo projetado para otimizar a produtividade durante sessões de estudo, utilizando a técnica Pomodoro. O projeto permite ao usuário alternar entre tempos de foco e pausas curtas ou longas. Além disso, o usuário pode ativar música de fundo para melhorar a concentração durante o foco. O design é simples e visualmente agradável, c om imagens dinâmicas que refletem o estado atual (foco ou descanso). A funcionalidade inclui contagem regressiva, botões para alternar os estados e a capacidade de pausar e reiniciar o cronômetro.

Lista de Compras Interativa

A aplicação permite inserir itens em uma lista de forma dinâmica, permitindo a exclusão e edição dos mesmos. Os itens terão o nome e também a data e horário que foram criados. Também é possível selecionar um checkbox para informar que o item foi comprado.

O objetivo deste projeto é criar uma ferramenta interativa e fácil de usar para ajudar os usuários a gerenciar suas compras e forma organizada. Com funcionalidades como adicionar, editar, excluir e marcar itens como comprados, o projeto visa tornar o processo de criação e controle de listas de compras mais eficiente e prático. A ideia é proporcionar uma experiência agradável e intuitiva, onde o usuário tem total controle sobre os itens e pode visualizar em tempo real o que precisa comprar e o que já foi adquirido.

I.D.O.S.ON

O Idoson é um projeto voltado para facilitar a inclusão dos idosos no mundo digital. O site foi desenvolvido com uma interface intuitiva, pensada especialmente para que os idosos possam navegar com facilidade. Nele, os usuários encontram diversas dicas práticas que auxiliam na adaptação ao uso da tecnologia.

Um dos principais recursos do Idoson é uma inteligência artificial que analisa o perfil de cada idoso com base nas respostas de um formulário disponível no site. A partir dessa análise, a IA oferece recomendações personalizadas, ajustadas ao nível de conhecimento tecnológico do usuário. Além disso, o site conta com um chatbot interativo que permite aos idosos esclarecerem dúvidas, e o chatbot também possui comandos de voz, proporcionando uma experiência ainda mais acessível.

Book-Store

O objetivo deste projeto foi criar uma aplicação simples de consulta e filtragem de livros, que consome dados de uma API externa para oferecer uma experiência interativa ao usuário. A aplicação foi desenvolvida com o intuito de aprimorar as habilidades de consumo de APIs e manipulação do DOM com JavaScript.

JúniorBooks é uma aplicação web que lista livros de tecnologia nas áreas de Front-End, Back-End e Dados. O projeto permite ao usuário filtrar os livros por categoria, visualizar os preços e ordenar os livros por preço. A aplicação consome dados de uma API pública e exibe informações sobre os livros, como título, autor, preço e categoria. O design é simples e funcional, com uma navegação intuitiva e rápida..

VidFlow

O VidFlow é uma réplica simples do YouTube, criada para estudar o consumo de APIs em JavaScript. O projeto busca dados de vídeos de um arquivo JSON hospedado localmente e os exibe de forma dinâmica na página. Ele permite filtrar os vídeos por pesquisa ou por categoria. O objetivo deste projeto é praticar a manipulação de dados de uma API externa e a criação de filtros dinâmicos para a interface.

Neste projeto, o foco foi aprender a fazer requisições a uma API externa e manipular os dados retornados. O código utiliza o método fetch() para buscar os dados em formato JSON e os exibe de forma dinâmica no HTML. Também inclui filtragem de vídeos tanto por pesquisa quanto por categoria, permitindo interação com o conteúdo e maneira fluida.

Sound-Pad

Este é um aplicativo interativo de Sound Pad, criado para estudar e praticar manipulação do DOM em JavaScript. O usuário pode tocar diferentes efeitos sonoros ao clicar em botões ou pressionar teclas de atalho, como a barra de espaço ou Enter. O projeto demonstra como interagir com elementos HTML, adicionar eventos e manipular atributos e classes de forma dinâmica.

Este projeto foca no estudo do DOM e sua manipulação com JavaScript: Seleção de Elementos: Usando document.querySelector e document.querySelectorAll para selecionar os botões e associar sons aos mesmos. Eventos: Inclusão de eventos de clique (onclick) e de teclado (onkeydown e onkeyup) para interagir com os elementos. Manipulação de Atributos e Classes: Modificação dinâmica das classes e controle de reprodução de áudio.
